### Option 2:
Building Docker file  
```
docker build -t vuejs-cookbook/dockerize-vuejs-app .
```
Running Docker file  
```
docker run -it -p 8080:8080 --rm --name dockerize-vuejs-app-1 vuejs-cookbook/dockerize-vuejs-app
```
